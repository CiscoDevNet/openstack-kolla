Prepare your our machine (e.g. your personal laptop) for use with this lab.

Download OVA file

  • For this lab you will need the "ciscolive_kolla_mitaka.ova" file created for this lab.
  • An OVA(Open Virtualization Appliance) package is a means of distributing software to be run in virtual machines. Download this file and store it in a convenient place on your laptop. We will use it latter.
  • Download the file here: ciscolive_kolla_mitaka.ova

Disable any VPN

  • Everything needed to complete this lab is available via the public network. Running a VPN may interfere with the local routing needed to access the OpenStack environment from your laptop. If possible, disable any VPN for the remainder of this lab.

Install VirtualBox [Optional]

  • This lab requires some means of starting and managing virtual machines. The examples in this lab are based on VirtualBox (version 5.0.20 r106931). You can use something else if you prefer.
  • VirtualBox download images and instructions are available at https://www.virtualbox.org/wiki/Downloads
  • You should install VirtualBox, followed by the VirtualBox extension pack. Note that in addition to the installer itself, the extension pack should be installed after the base VirtualBox platform is installed, but prior to starting any VMs.
  • Additional information is available at: http://virtualbox.org
